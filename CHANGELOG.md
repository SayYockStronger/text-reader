# Change Log

All notable changes to the "text-reader" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.1] - 2026-04-21

### Added

- サイドバーのファイルエクスプローラーからテキストファイルを選択して読み上げ
- kuromoji.js による漢字→カタカナ変換で読み上げ精度を向上
- 音声選択機能（OS にインストールされた日本語音声から選択）
- 読み上げ速度調整（1.0x ～ 3.0x）
- 音量調整スライダー
- 一時停止・再開・停止
- カーソル位置からの読み上げ
- キーボードショートカット対応
- インストール/更新時のリロード通知
